Dell N5010安装Yosemite(传统bios+GPT,可升级到EI CAPITAN)

发表于 2016-04-18   |     |   阅读次数

前言

上大学购置的第一台笔记本DELL N5010,戴尔的质量可是杠杠的

电脑型号 戴尔 Inspiron N5010 笔记本电脑
操作系统 Yosemite10.10.5
处理器 英特尔 Core i3 M 380 @ 2.53GHz 双核笔记本处理器
主板 戴尔 0YXXJJ (英特尔 HM57 Express Series 芯片组)
内存 4+2 GB ( 海力士 DDR3 1333MHz )
主硬盘 希捷 ST9500420AS
显卡 ATI Mobility Radeon HD 5650 ( 1 GB / 戴尔 )
无线网卡 BCM4322 淘宝购入

OK.Let’s do it!

准备工作

制作启动盘

  • 两个U盘,一个安装clover EFI引导(备用),一个用来写入安装系统
  • 使用BootDiskUtility,写入默认的Clover引导(开机使用U盘驱动,然后选择Boot install mac在安全模式verbose boot下可以引导起来Yosemite,无须设置)

BootDiskUtility 使用教程

制作系统盘

  • 系统可在远景论坛下载,建议下载原版系统(我用的是自带EFI的版本),使用TransMac写入即可,放在一边备用

    驱动HD5650显卡的准备工作(关键)

  • 在win7下使用EDIDViewer400,导出你显示器屏幕的EDID,保存成文本备用

如果是同款机型,可以在win7上安装MAC OS的虚拟机,然后直接在虚拟机里使用chn_DSDTSE把DSDT里面的EDID替换成你自己提取的,然后选择编译,完成后会弹出一个文件夹,保存备用.

DSDT更改EDID教程
ATI笔记本显卡内屏senseid为07的解法 5XXX和6XXX适用

正式安装

完成引导开机

  1. 插入U盘引导,开机按F12
  2. 选择USB引导模式启动,在选择界面插入系统盘
  3. 看见BOOT MAC INSTALL等字样的,按下空格键,选择verbose boot启动,回车选择

ps:系统验证的问题:如果你用的是带EFI的景象,那么会提示无法验证此镜像,不用怕,只要在命令行里输入:
date 041302022015
然后关闭命令行即可,就可以继续安装了,这个的原理是修改到2015年4月13,就能通过验证了.

分区安装

  1. 选择磁盘工具先格式化硬盘,格式选择日志扩展式,在磁盘分区上选择GPT分区模式,点击抹盘,等待完成即可
  2. 点击下一步,会提示一个意见信息,直接点击Agree
  3. 到达安装盘符的界面,选择你刚刚格式好的分区,选择,开始安装(大概需要20分钟,在20分钟后会提示安装成功,需要重启,确认即可)
  4. 重启过程中重复安装引导的步骤,从引导进入,进入第二次正式安装
  5. 如果一切顺利的话,等待下次重启就可以拔掉系统盘了

EFI写入分区

  1. 把clover EFI引导安装在磁盘EFI分区中
  2. 使用EFI CLOVER TOOLS工具,先格式化EFI分区,然后安装EFI clover
  3. 选项如下图:


安装完成会多一个EFI分区

安装详情参考【混脸熟（第二季）】Dell n5010 Yosemite完美驱动99.9%无痛升级历程分享

完善EFI

  1. 完成后,把之前已经配置好的config.plist替换,安装相应驱动,然后把你编译好的DSDT,放入ACPI/PATCHED文件中,重启
  2. 重启的过程中可以拔出引导U盘了,然后就是激动人心的一刻了
